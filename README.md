# Product Search Engine

A web application that allows users to search for products across multiple e-commerce platforms simultaneously. This project focuses on creating a seamless experience for comparing products and prices from various online retailers.

## üåü Features

- **Multi-platform Search**: Search across major e-commerce platforms including Amazon, Flipkart, Meesho, Myntra, Ajio, BuyHatke, MySmartPrice, and Google Shopping
- **Dynamic Product Images**: Automatically loads relevant product images based on search terms using Unsplash API
- **Automatic Spell Correction**: Corrects common misspellings in search queries for better results
- **Horizontal Scrolling Interface**: Platform links organized in a clean horizontal layout with navigation arrows
- **Direct Actions**: Open product pages directly or copy URLs to clipboard
- **Bulk Link Download**: Download all generated search links as a text file
- **Responsive Design**: Works seamlessly across desktop, tablet and mobile devices
- **Professional Footer**: Includes developer attribution and LinkedIn profile link

## üöÄ Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (Apache, Nginx, or any static file server)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/product-search-engine.git
   ```

2. Navigate to the project directory
   ```bash
   cd product-search-engine
   ```

3. Start a local web server
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

4. Open your browser and go to `http://localhost:8000`

## üí° Usage

1. Enter a product name in the search box
2. Click "Search" or press Enter
3. View the dynamically loaded product image
4. Browse through platform options using the horizontal scroll arrows
5. Click "Open" to visit the product page on a specific platform
6. Click "Copy" to copy the URL to your clipboard
7. Use "Download all links" to save all URLs to a text file
